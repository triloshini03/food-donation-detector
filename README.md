# Food Donation Detector

This project is a prototype system that detects food freshness and helps reduce food wastage.

## Features
- Upload food image
- Detect whether food is fresh or spoiled
- Simple user interface

## Technologies Used
- Python
- Streamlit
- Image Processing

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## Project Status
Prototype version. Future work includes real AI model integration.

## Future Scope
